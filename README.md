# LifeShare - Organ Donation & Hospital Coordination System

## Project Overview
LifeShare is a web-based organ donation management system that connects donors, hospitals, patients, and administrators through a centralized platform.

## Features
- Donor Registration & Login
- Hospital Registration & Login
- Organ Request Management
- Emergency Request Notification
- Email Alerts
- Donor Profile Management
- Request Approval & Rejection
- Admin Dashboard

## Technologies Used
### Frontend
- React JS
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask

### Database
- MySQL
